# The Little Puppet Boy

This game was created in 48 hours, for [Global Game Jam 2026](https://globalgamejam.org/games/2026/little-puppet-boy-0), with the theme of "Mask". [You can play on itch.io here.](https://sirlich.itch.io/the-little-puppet-boy).
